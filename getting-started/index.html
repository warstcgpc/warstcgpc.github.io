Getting Started

by JS
published on February 28, 2026

Welcome!

This is a small start at making a centralized Players Committee launch.  At this time, it's just myself.  I'm hoping to coordinate with a few of the folks across Facebook and Discord to align everyone's projects and strengths so that we all move forward as a group.

I'll work on updating resources slowly to get new players up and running.

I was able to update the awesome database from matthalm on discord (matthalm/warsdb: A barebones filterable database for Decipher's WARS TCG) and update Set 3 with the fan completed Edge of Sword set that Gady on Facebook made.

An initial Google Map Pin player locator has been created as well!
